What are the ways that an agent can be hijacked?

 - Prompt Injection: attacker includes malicious text in the supplied prompt.
 - Context Injection: attacker includes malicious text in web documents, files, or other context sources.
 - Memory Poisoning
 - Tool Misuse
 - Data Exfiltration
